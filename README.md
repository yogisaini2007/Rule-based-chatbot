# A Rule-Based Chatbot

A lightweight Python chatbot that responds to basic user input through a simple rule-matching approach. This project is designed as an introductory implementation of a conversational agent powered by conditional phrase matching.

## Features

- Rule-based reply generation for common greetings and questions
- Normalizes user input for consistent matching
- Clean command-line interaction with a friendly chatbot persona
- Easy to extend with additional response rules

## Getting Started

### Prerequisites

- Python 3.6 or later

### Installation

1. Clone or download the repository.
2. Open a terminal in the project folder.
3. Run the chatbot using Python:

```bash
python chatbot.py
```

## Usage

Type a message and press Enter. The chatbot currently supports simple inputs such as:

- `hello`, `hi`, `hey`, `hii`
- `how are you`
- `who are you`
- `help`
- `what is your name`
- `bye`, `exit`

To quit the chatbot, type `bye` or `exit`.
